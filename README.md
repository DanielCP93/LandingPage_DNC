# Minha Primeira Landing Page - DNC School! 🚀

Olá! Sou um desenvolvedor front-end iniciante e este é meu primeiro projeto. Criei esta landing page durante meus estudos na Escola DNC. Foi um desafio bem legal e aprendi muito!

🔗 **Veja o projeto funcionando:** [GitHub Pages](https://danielcp93.github.io/LandingPage_DNC/)

## Sobre o Projeto

Esta é uma página de captura de leads para o curso gratuito de Desenvolvedor Front-end da DNC School. A página tem várias seções legais e funciona bem em celular, tablet e computador.

### O que a página tem:
- **Cabeçalho com logo**: Link direto pro site da DNC
- **Informações do curso**: Mostra que tem certificado, 4 horas de aula e aulas gravadas
- **Vídeo do YouTube**: Apresentando a DNC.
- **Depoimentos**: Carrossel com comentários de alunos (Leonardo, Samantha e Bruna)
- **Formulário**: Pra se inscrever no curso (nome, email, telefone)

## Tecnologias que usei

- **HTML**: Estrutura da página
- **CSS**: Estilos e responsividade
- **JavaScript**: Funcionalidades do carrossel e formulário

## Como funciona

### Responsividade
A página se adapta para diferentes tamanhos de tela:
- Celular pequeno (até 479px)
- Celular normal (480px-767px)
- Tablet (768px-1023px)
- Desktop (1024px+)

### JavaScript
- **Carrossel**: Botões para navegar entre os depoimentos
- **Formulário**: Quando clica em "Acessar detalhes do CURSO", abre o site da DNC

## Estrutura dos arquivos

```
LandingPage_DNC/
├── index.html          # Página principal
├── style.css           # Todos os estilos
├── index.js            # Funcionalidades
├── cards/              # Imagens dos depoimentos
├── icons/              # Ícones (setas, certificado, etc.)
└── image/              # Logo e fundo
```

## Como testar

1. Baixe/clone este repositório
2. Abra o arquivo `index.html` no navegador
3. Teste em diferentes tamanhos de tela (use F12 > responsivo)
